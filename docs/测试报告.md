# 测试报告

## 测试范围

本轮测试目标覆盖三层：

- 单元测试：验证后端持久化工具的正确性（JSON 读写与回退）。
- 前端 UI 单元测试：验证“查看规范/删除规范”按钮交互、异常与重试埋点。
- 集成测试：启动后端服务后对核心 API 进行协议级验证（HTTP 状态码、返回结构、关键字段）。
- 端到端（烟雾）：在存在 `dist/` 构建产物的前提下验证服务端对 SPA 路由的回落能力（`/spec-data` 返回 `index.html`）。

## 测试入口与命令

- 全量：`npm test`
- 分层：
  - 单元：`npm run test:unit`
  - 前端 UI：`npm run test:ui`
  - 前端 UI 覆盖率：`npm run test:ui:coverage`
  - 集成：`npm run test:integration`
  - 端到端（烟雾）：`npm run test:e2e`

脚本定义见：[package.json](file:///h:/learnproject/04%20solo_builder/package.json)
